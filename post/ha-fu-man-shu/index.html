哈夫曼树

发布于 2020-04-19 | 1分钟 | 178字数 | 浏览量：

哈夫曼编码有等长和不等长之分。

等长

nnn个编码最多可以放2n2^n2n个字符。

不等长

要求：

  • 频率越高，长度越短；
  • 一个字符编码不能是另一个字符编码前缀，即不能有二义性；
  • 编码只有000和111.

哈夫曼树

假如说现在有一个字符串aaabbbbbbbbbbbbbbbbbbbbbbbbbbbbbbcdddfee

则里面的a有333个，b有303030个，c有111个，d333个，e222个，f111个。

则我们编码规则左000右111。

那么我们先去两个最小的字符c,f，构成节点如下所示：
c1 f1，合并得

 []2
 /  \
c1  f1

再选一个最小的：

   []4
  /   \
 []2   e2
 /  \
c1  f1

递归即可。

成果：

         []40
        /    \
       []10   a30
      /   \
     []7   d3
    /   \ 
   []4   a3
  /   \
 []2   e2
 /  \
c1  f1
